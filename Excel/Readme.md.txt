# Proyecto: Ventas de Supermercado

Este proyecto forma parte de mi portafolio de análisis de datos.  
Incluye un archivo de Excel con información de ventas simuladas de un supermercado.  
Fue utilizado para aplicar funciones, tablas dinámicas y gráficos con el objetivo de analizar el comportamiento de las ventas.

**Archivo:** [Descargar VentasSupermercado.xlsx](./Excel/VentasSupermercado.xlsx)

**Temas aplicados:**
- Funciones de búsqueda y fecha en Excel.
- Tablas dinámicas con filtros y segmentación.
- Análisis de ventas por categoría, producto y sucursal.
- Creación de gráficos y dashboard de resumen.

Bienvenido/a a mi portafolio de **Análisis de Datos**.  
Aquí encontrarás ejemplos de proyectos y ejercicios desarrollados con diferentes herramientas y lenguajes, organizados para mostrar mi progreso y habilidades en el área.

Este portafolio está en construcción y se irá actualizando con nuevos proyectos a medida que avance en mi formación y experiencia.



Estructura del Portafolio

- **Excel/** → Dashboards, reportes y análisis de datos usando tablas dinámicas, gráficos y fórmulas.
- **PowerBI/** → Visualizaciones interactivas, reportes dinámicos y modelos de datos.
- **Python/** → Scripts para análisis de datos, limpieza, automatización y visualización (pandas, matplotlib, seaborn, etc.).
- **SQL/** → Consultas y análisis sobre bases de datos para extraer, filtrar y transformar información.
